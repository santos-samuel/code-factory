# Report Template

Use this template for the Step 5 report in `/skill-workbench`. Omit sections with no entries.

```markdown
## Improvements Made

### Skills Updated
- **{plugin}:{skill-name}**: {what changed and why}
  - Before: {brief excerpt}
  - After: {brief excerpt}

### Documentation Updated
- **{file}**: {what changed and why}

### Tools Improved
- **{tool}**: {what changed and why}

### New Skills Created
- **{plugin}:{skill-name}**: {what it does}

### Version Bumps
- **{plugin}**: {old} -> {new}

### Suggested Follow-ups
- {anything that needs deeper work via /do}
```
